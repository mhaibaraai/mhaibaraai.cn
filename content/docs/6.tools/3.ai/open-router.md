## OpenRouter 注册

![openrouter](/images/tools/ai/openrouter.png)

1. 访问 [OpenRouter](https://openrouter.ai/) 并注册一个账户。
2. 登录后，右上角头像导航到 "Keys" 页面。
3. 创建一个新的 API 密钥，并将其保存以备后用。

### 查找免费模型

点击 [Models](https://openrouter.ai/models) 页面，浏览可用的免费模型。

![openrouter-free-models](/images/tools/ai/openrouter-free-models.png)

::warning
OpenRouter 上有很多免费模型，需要注意的是这些模型速率限制较低（每天总共 50 次请求），通常不适合用于生产环境。可以选择充值 10 美元，以每天获得 1000 次请求。（免费模型）
::

![openrouter-free-tier](/images/tools/ai/openrouter-free-tier.png)

### 使用付费模型

点击右上角头像导航到 "Credits" 页面，选择 "Add Credits"，可以通过 “支付宝、微信” 付费充值

::warning
在使用 OpenRouter 购买积分时，会收取 5.5%（最低 0.80 美元）的手续费。不增加任何附加费用，直接转介底层模型提供者的定价，因此支付的费率与直接向提供者支付相同。
::

![openrouter-credits](/images/tools/ai/openrouter-credits.png)
